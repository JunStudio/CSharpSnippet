## 公众号二维码

知道一个微信公众号,怎么知道它的二维码是什么呢? - 知乎 - 知乎
2016年1月21日 - 

http://open.weixin.qq.com/qr/code/?username=

「=」之后键入公众号的「微信号」,就可以得到公众号的二维码图片。 发布于 2016-01-21 7 条评论 感谢 ...
